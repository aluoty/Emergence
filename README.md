# Emergence — Civilization Simulator

Build your civilization from a small settlement to a multiversal empire.
Manage resources, research technology, explore the stars, and interact with
alien civilizations.

## Building & Running

```bash
make           # build (requires raylib + deps)
./build/emergence
make clean     # remove binary
```

## Keybindings

| Key | Action |
|-----|--------|
| Q / W / E / R / A | Switch tab: Overview / Buildings / Tech / Civs / Achievements |
| H | Hunt (sends hunters for 2 days) |
| F | Farm (plant crops, harvest in 3 days) |
| G | Gift food to boost morale |
| B | Build homes (+5 pop cap) |
| U | Research (spend 5 knowledge over 3 days) |
| T | Trade (sell wood for gold; on Civs tab: trade with selected civ) |
| Z | Rest (recover health + morale) |
| C | Heal (use herbs) |
| M | Recruit (hire 2 new settlers) |
| S | Scout (explore for resources) |
| P | Train (train a soldier) |
| I | Colonize (launch a colony expedition) |
| Y | Build starship |
| O | Explore galaxy |
| K / L | Save / Load game |
| N | New game |
| 1-6 | Assign jobs: Farmer / Woodcutter / Miner / Hunter / Scholar / Priest |
| 0-9 | On Buildings tab: build structure; on Tech tab: research |

### Civs Tab

| Key | Action |
|-----|--------|
| 0-5 | Select a civilization |
| T | Trade with selected civ |
| W | Declare war on selected civ |
| A | Form alliance with selected civ |
| V | Switch view to selected civilization |

## Resources

- **Food** — consumed by population each turn (x2 in winter)
- **Wood, Stone** — used for construction
- **Gold** — currency for trade, recruiting, training
- **Herbs** — used for healing
- **Knowledge** — spent on technology research
- **Faith** — generated by priests and temples

## Jobs

Assign idle population to jobs (1-6 keys):
- **Farmer** — produces food
- **Woodcutter** — produces wood
- **Miner** — produces stone
- **Hunter** — produces food (can be risky in winter)
- **Scholar** — produces knowledge
- **Priest** — produces faith

## Technologies

14 technologies across the tech tree. Key ones:
- **Agriculture** — farms +50%
- **Mining** — mines +50%
- **Carpentry** — woodcutters +50%
- **Medicine** — health x2
- **Astronomy** — unlocks colonization
- **Propulsion** — unlocks starship construction
- **Xenology** — communicate with aliens
- **Transcendence** — ultimate knowledge

## Tiers of Civilization

1. **Planetary** — build on your home world
2. **Interplanetary** — colonize other planets
3. **Interstellar** — explore the galaxy
4. **Universal** — manipulate space and time
5. **Multiversal** — transcend reality itself

## Achievements

36 achievements tracking milestones from first settlement to multiversal
dominance.

## Victory

Reach 100 population or survive 500 turns.
